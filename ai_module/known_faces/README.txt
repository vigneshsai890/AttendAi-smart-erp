This folder is for storing reference images of known students for face recognition.

1.  **Add Your Photos:**
    Place clear, front-facing images of the students you want to recognize in this folder.
    
2.  **Naming Convention:**
    Name the files exactly to match the `faceEncodingId` in your database.
    For example:
    - If `face_001` is John Doe, name the image `face_001.jpg` (or `.png`).
    - If `face_002` is Jane Smith, name the image `face_002.jpg`.

The AI module will load these images on startup and learn to recognize these faces!
